# Cortex Clipper — Chrome Extension

## Install (Developer Mode)
1. Open Chrome → `chrome://extensions`
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the `src/extension/` folder
5. The Cortex Clipper icon appears in your toolbar

## Usage
1. Navigate to any page you want to clip
2. Click the Cortex Clipper icon
3. Optionally add a note
4. Click "Clip to Cortex Inbox"
5. The link appears in your Cortex inbox immediately

## Requirements
- Cortex must be running locally (`npm run dev` or `npm start`)
- Cortex must be on port 3001
